# POB 쇼핑몰
2013년 제작.  
모바일 스피커 케이스인 POB 의 쇼핑몰 사이트다.  
사내 프레임워크 2.0을 기반으로 제작 되었으며 media-query를 이용하여 모바일 및 태블릿 기기에도 잘 보일 수 있도록 하였다.

## 사용기술 및 환경
* Java 1.6
* Struts2 & myBatis
* jQuery & jQuery-UI
* MySQL 5.5
* CSS3: media query 적용
* 카드 및 계좌이체등 결제 모듈 적용

## 비고
사정상 보안에 민감한 내용과 하부 컨텐츠 및 SQL문 등은 제거 함.

## Caution
- 내용을 참고만 하되 복붙하심 안됩니다..
